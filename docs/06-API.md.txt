# Lesson 16
## Endpoint : GET /api/v1/products/{id}
Purpose: Returns one product by ID.
Status: Implemented
Response: 200 OK

GET /api/v1/products/{id}
Responses: 
200 OK
404 Not Found

POST /api/v1/products
Validation Rules: 
- Name required
- Description required
- Price > 0
- Quantity >= 0
Responses: 
201 Created
400 Bad Request

### Validation Error Response
Status: 400
{
  "timestamp": "...",
  "status": 400,
  "message": "Validation Failed",
  "errors": {
      "field": "validation message"
  }
}

## Update Product
PUT /api/v1/products/{id}
Request
{
  "name": "...",
  "description": "...",
  "price": 100,
  "quantity": 5
}
Response : 200 OK
Returns ProductResponseDTO

## Delete Product
DELETE /api/v1/products/{id}
Response : 204 No Content

### API Design
Request: Client → ProductRequestDTO
Response: Server → ProductResponseDTO
Controllers never expose JPA entities directly

### Get Products (Paginated)
GET /api/v1/products?page=0&size=10
Query Parameters :page, size
Returns : Page<ProductResponseDTO>
