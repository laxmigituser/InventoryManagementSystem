# Glossary

## Maven

A build automation and dependency management tool for Java projects.

---

## POM

Project Object Model.

The XML file that tells Maven how to build the project.

---

## Dependency

An external library your application uses.

Example:

Spring Web

---

## Artifact

The output produced by Maven.

Usually a JAR file.

---

## Snapshot

A development version of an application.

## Starter Dependency

A Spring Boot dependency that brings in a collection of related libraries.

Example:
spring-boot-starter-web

---

## Lombok

A library that generates boilerplate code such as getters, setters, constructors, builders, and more during compilation.

---

## DevTools

A Spring Boot library that automatically restarts the application when code changes during development.
